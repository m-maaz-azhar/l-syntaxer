## Live @ [L-Syntaxer](https://l-syntaxer.vercel.app/)

## Features

- [x] C# to Python
- [x] Python to C#
- [x] Language detection ( for now: C# / Python)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
